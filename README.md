Αν σας ενδιαφέρει η ιστορία και ο πολιτισμός της Κύπρου, σας προσκαλώ να επισκεφθείτε τον χώρο μου.
Εδώ παρουσιάζονται τόποι, μνημεία και στοιχεία της κυπριακής παράδοσης, τεκμηριωμένα αποκλειστικά με τη δική μου φυσική παρουσία σε κάθε σημείο, μέσα από επιτόπια βιντεογράφηση και φωτογράφιση.
Στόχος μου είναι η αυθεντική καταγραφή και ανάδειξη της Κύπρου, όπως πραγματικά είναι.
